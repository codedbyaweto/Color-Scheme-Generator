# Color Scheme Generator(colorgenerato.netlify.app)

A simple and clean color scheme generator that lets you pick a base color and instantly view matching color palettes. The app uses **The Color API** to fetch color schemes such as Monochrome, Analogic, Triad, Complement, and more.

The generated colors are displayed as **vertical side-by-side columns**, each showing its HEX code below the bar.

---

## Features
- Pick any color using a native color picker.
- Choose from multiple scheme types (monochrome, analogic, triad, complement, etc.).
- Fetches color palettes from **The Color API**.
- Displays 5 colors as **full-height vertical columns**.
- Clean, modern UI inspired by professional palette tools.
